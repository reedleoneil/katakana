术语“名词”是神秘的，并且极易引起误解。日本的正式名称是Keiyou-doushi，字面意思是“形容词动词”，但广名名词绝对不是动词。交际名词不能像他们的大哥哥（真正的形容词）那样完成句子。必须使用交际动词来完成句子，其中不带NA附件的交际名词是补语。 cop名词也被错误地称为“ na-形容词”，因为它们带有NA附件可以像[Copular Noun + NA + Noun]中那样修饰名词，就像带有NO附件的名词一样，例如[Noun + NO] +名词]。 coppular名词是一类特殊的名词，不是形容词。正如上一章所回顾的那样，正是系动词NA的定语基础形式，使系名词可以修饰其后的名词。但是，建议记住保留NA完整的交际名词以避免混淆，因为它们总是被翻译成英文的形容词。在这本语法书中，与某种形容词相反，俗语名词被视为一类特殊的名词，并被缩写为CN。为了指示NA的存在或不存在，该语法书使用以下符号：CN + NA：其中上标指示NA的存在。 CN + 0：其中上标指示不存在NA。补语名词作为修饰语。齿状名词作为代名词的组成部分：[CN + NA + NO]。补语名词作为补语动词的补充：[CN + 0 + DESU]。作为副词成分的交配名词：[CN + 0 + NI]。 [CN + 0 + NI]成分首先在第3.1.1.2节：不完整的不及物动词中引入。回想一下，链接动词要求其句子的互补元素具有最小的意义。这些补语部分地归因于连接动词的功能，在日语中是副词，表示动词的语义性质内的主语的质量或特征。换句话说，交际名词可以形成副词成分。在下面的例句中，平假名句子中用下划线标出了这些副词。在罗姆族句中，用括号括起来。例句（d）和（e）证明了副词成分[CN + 0 + NI]作为连接动词的补充。例句（f）和（g）展示了更直接的副词功能，描述了动作的方式和质量或动作的结果。下面的所有例句都证明了副词成分在功能和外观上的差异和相似性。示例（d）至（g）中的状语成分是基于交际名词的。在示例（h）中，副词成分是衍生自形容词的副词，在（i）中，副词成分是基于名词的。这种比较突显出交际名词的确更像名词，并且与形容词完全不同。副词修饰了Copular名词。不是形容词。口语名词源自其他词性。外国形容词的名词。日语在语法，语义和形态上都具有灵活性。每天都会添加新词，尤其是在网络空间中。这不是一个新现象。相反，这是日本文化的模式。他们采用，内部化和完善从外部世界获得的东西。日本人首先从中国获得了书写系统。日语然后改编了中国起源的活性名词化合物。最后，采用西方词语。日语中已经有很多外来词。单词被同化的方式是特殊的。在修饰紧随其后的名词时，外来形容词往往被同化为带有签名摩拉的交配名词。交配名词永远不会成为动词。识别鼻翼名词。如果很难识别一个流行名词，请在句子中的主语，直接宾语和间接宾语位置使用不带NA附件的相关单词。如果该单词适合作为主语，直接宾语，间接宾语或它们的任意组合，则很可能不是流行名词。或者，尝试用可属粒子NO代替NA修饰名词。如果这可行，则该单词不是反义名词。将单词分为一个类别或另一个类别确实不是一个明确的决定。下面的示例令人困惑，因为它们具有形容词含义，但它们是某种名词。
